# Sales Analysis and Dashboard Project (Excel)

## Project Overview
This project involves analyzing sales data and building an interactive dashboard using Microsoft Excel.

The focus was on calculating key business metrics and presenting insights through structured reporting.

---

## Dataset Description
The dataset contains sales transaction records including product details, dates, and financial data.

---

## Data Preparation

- Cleaned and structured the dataset for analysis
- Extracted **month values from date fields** using Excel text functions

---

## KPI Development

The following key performance indicators (KPIs) were calculated:

- Revenue
- Cost of Goods Sold (COGS)
- Profit

These were derived using Excel formulas to enable performance analysis.

---

## Data Analysis

- Used **Pivot Tables** to summarize and analyze sales data
- Identified trends and performance patterns across different dimensions

---

## Dashboard Creation

- Built a dashboard to visualize:
  - Revenue trends
  - Profit performance
  - Sales distribution

---

## Key Insights
- Enabled clear visibility into sales performance
- Highlighted profit trends and revenue drivers
- Provided a structured view for decision-making

---

## Tools Used
- Microsoft Excel
- Pivot Tables
- Data Visualization (Charts)
- KPI Calculations

---
